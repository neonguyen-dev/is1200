First opt to MCBTools using MSYS terminal or another equivalent terminal by issuing the command:

. /opt/mcb32tools/environment

If mcb32tools is not installed, follow the instructions on http://github.com/is1200-example-projects/mcb32tools/releases/

Navigate to the directory of the project folder using cd commands. Connect your chipKIT to the workplace and
issue the commands in MCB32tools:

make

make install
